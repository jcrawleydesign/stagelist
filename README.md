
# Functional Prototype for Songs

This is a code bundle for Functional Prototype for Songs. The original project is available at https://www.figma.com/design/u4I0DHq0Eox7S4myVBwjkA/Functional-Prototype-for-Songs.

## Getting Started

### Installation

Run `pnpm install` to install the dependencies.

### Development

Run `pnpm dev` to start the development server.

### Building

Run `pnpm build` to create a production build in the `dist` directory.

Run `pnpm preview` to preview the production build locally.

## Deployment

For detailed deployment instructions, see [DEPLOYMENT.md](./DEPLOYMENT.md).

Quick deployment options:
- **Vercel**: Connect your repository and deploy automatically
- **Netlify**: Import your repository and deploy
- **Azure Static Web Apps**: Use Azure Portal or CLI
- **GitHub Pages**: Use the `gh-pages` package

All deployment configuration files are already set up in this project.
